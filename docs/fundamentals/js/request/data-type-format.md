# 数据类型与格式

## 数据类型

> There are many kinds of data structures, but they can all be adequately represented with three basic primitives: mappings (hashes/dictionaries), sequences (arrays/lists) and scalars (strings/numbers). 

* 标量（scalar）
* 序列（sequence）
* 映射（mapping）
