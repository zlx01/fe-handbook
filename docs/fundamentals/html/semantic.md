# Semantic

## why

1. 根据内容的结构（内容语义化），选择合适的标签，便于开发者阅读理解代码。
2. 同时让浏览器的爬虫很好地解析，有利于SEO

```html
<body>
  <header>页眉</header>
  <main>
    <article>
      <h1>文章标题</h1>
      <p>文章内容</p>
    </article>
  </main>
  <footer>页尾</footer>
</body>
```

## tags

* header
* main，只能有一个
* footer
* aside
* article
* section
* nav
* h1~h6
