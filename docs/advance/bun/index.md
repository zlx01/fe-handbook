# [Bun](https://bun.com/)

## all-in-one toolkit

- runtime
- package manager
- bundler
- test runner

## built on

- JavaScriptCore
- Zig

## language support

- JavaScript
- TypeScript
- JSX
- TSX

## cli

```bash
bun run <script>

bun install <package>

bun test

bun build <entry>
```

## bunx

replace `node` and `npx` with `bunx` to run commands from `node_modules/.bin`

```bash
bunx <command>
```

```bash
bunx cowsay 'Hello, world!'
```
