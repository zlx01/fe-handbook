HTTP Caching #

  • 强缓存 Strong cache
  • 协商缓存 Negotiation cache

HTTP协议中规定使用GMT时间，也就是格林威治标准时间，而我们国家使用的是GMT+8时区，所以在HTTP头信息中的时间会比我们的正常时间早8个小时，但它一点都不影响HTTP缓存的正常工作。
